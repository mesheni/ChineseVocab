# ChineseVocab

**ChineseVocab** — это кроссплатформенное мобильное приложение для изучения китайских иероглифов, построенное на базе .NET MAUI. Проект использует систему интервальных повторений (SRS) для оптимизации процесса запоминания и ориентирован на эффективную подготовку к экзаменам HSK.

## 🚀 Основные возможности (в разработке)

  * **Алгоритм SM-2**: Интеллектуальное планирование повторений на основе ваших ответов.
  * **Специфика китайского языка**: Поддержка пиньиня, озвучка (TTS) и визуализация порядка начертания черт.
  * **Режим диктанта**: Тренировка написания и распознавание речи.
  * **Персонализация**: Создание собственных колод и импорт/экспорт данных.
  * **ИИ-помощник**: Генерация мнемонических историй для упрощения запоминания сложных иероглифов.

## 🛠 Технологический стек

  * **Фреймворк**: .NET 8 / MAUI (Multi-platform App UI)
  * **Язык**: C\#
  * **Архитектура**: MVVM (Model-View-ViewModel)
  * **Библиотеки**:
      * `CommunityToolkit.Mvvm` — для реализации логики уведомлений и команд.
      * `SQLite-net-pcl` — для локального хранения данных.
      * `SkiaSharp` — для отрисовки иероглифов.

## 📈 План разработки

Проект разделен на 11 этапов. Текущий статус: **Этап 1 (Завершен)**.

1.  **Инфраструктура**: ✅ Настройка DI, навигации и базовых классов.
2.  **База данных**: Проектирование схем SQLite и CRUD операций.
3.  **Движок SRS**: Реализация алгоритма SM-2 и логики планирования.
4.  **UI Обучения**: Интерфейс карточек (Flashcards) и анимации.
5.  **Китайский модуль**: Порядок черт, тоны и примеры предложений.
6.  **Диктант**: Интеграция Cognitive Services для распознавания речи.
7.  **Управление**: Библиотека иероглифов, фильтрация по уровням HSK.
8.  **Аналитика**: Графики прогресса и экспорт статистики.
9.  **AI-интеграция**: Генерация контента через API.
10. **Оптимизация**: Unit-тесты и профилирование производительности.
11. **Релиз**: Локализация и подготовка к публикациям в сторах.

## 📂 Структура проекта

  * `Models/` — сущности данных (Card, Deck, SRSStat).
  * `ViewModels/` — логика интерфейса.
  * `Views/` — страницы приложения (XAML).
  * `Services/` — бизнес-логика (SRS, Database, Statistics).
  * `Resources/` — шрифты, изображения и стили.

## ⚙️ Установка и запуск

1.  Установите [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).
2.  Установите рабочую нагрузку MAUI:
    ```bash
    dotnet workload install maui
    ```
3.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ваш-аккаунт/chinesevocab.git
    ```
4.  Откройте решение в Visual Studio 2022 или VS Code.
5.  Выберите целевую платформу (Android, iOS или Windows) и запустите проект (F5).
